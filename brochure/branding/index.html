Branding Guide

For Designers and Developers

Welcome to the resource for AllPlayers.com branding guidelines. Here you can find various styled elements for your next AllPlayers related project.

For Designers

This resource is intended to be a guide for designers when theming or styling everything AllPlayers. Built with SASS, the styles for this branding guide are meant to be simple to implement.

For Developers

Not sure how to format certain elements for AllPlayers aplications? This branding resource contains excerpts and code snippets of "dream markup". By ensuring the proper markup, applications can retain consistant style across the board.
