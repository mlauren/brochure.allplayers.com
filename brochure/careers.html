Careers

Join our team of innovators

At AllPlayers.com we enjoy a casual team environment with many non-traditional benefits offered to all employees. AllPlayers respects and empowers our intelligent team of Drupal developers, user interface designers and member services leaders to enhance skills and life experiences collaboratively as we improve the quality of our communities.

Drupal / PHP Web Developer

  • Experience developing in Drupal 6
  • 4-6 years experience working with PHP,MySQL, Javascript, AJAX, HTML, CSS
  • In depth experience with Drupal 5/6 modules, Drupal Module development
  • Web services experience
  • Able to perform in a fast-paced startup environment
  • Familiar with version control systems such as SVN or GIT

Information Architect / UI

  • Develop thorough, realistic plans that support organizational objectives
  • Work with customers to understand their business models and goals and help define strategy, content, and features for design of their web site
  • Analyze audiences and their information and functional needs
  • Define site architecture and navigation that serves as a blueprint of the site upon which all other aspects are built
  • Create wireframes, site maps, schematics, process maps, feature lists, mockups, visual specification, working prototypes and other artifacts to describe the intended user experience

Member Services Account Representative

  • Demonstrated sales and/or support success in a technology environment
  • Organizational skills with ability to multi-task
  • Passion for sports or other recreational activities
  • Understands and embraces the world of social networking
  • Ability to make contacts, set appointments, demonstrate our site and willingness to do whatever it takes to make our customers love the AllPlayers platform as much as we do
  • In depth knowledge of excel or willing to learn

MySQL Database Administrator

The database administrator (DBA) will be responsible for the performance, integrity and security of the AllPlayers.com databases including:

  • Establishing the needs of users and monitoring user access and security
  • Monitoring performance and managing parameters to provide fast query responses
  • Mapping out the conceptual design for a planned database in outline
  • Controlling access permissions and privileges
  • Developing, managing and testing back-up and recovery plans
  • Ensuring storage, archiving, back-up and recovery procedures are functional
  • Capacity planning
  • Communicating regularly with staff to ensure database integrity and security

Systems Administrator

  • Perform daily system monitoring, verifying the integrity and availability of all hardware, server resources, systems and key processes, reviewing system and application logs, and verifying completion of scheduled jobs such as backups
  • Perform regular security monitoring to identify any possible intrusions
  • Perform daily backup operations, ensuring all required file systems and system data are successfully backed up
  • Perform regular file archival and purge as necessary
  • Create, change, and delete user accounts per request
  • Repair and recover from hardware or software failures
  • Perform periodic performance reporting to support capacity planning
  • Perform ongoing performance tuning, hardware upgrades, and resource optimization as required

Web Designer / Themer

  • Take mockups and other designs and apply them to the web site
  • Ensure web site support various browsers and versions adequately
  • Analyze audiences and their information and functional needs
  • Create wireframes, site maps, schematics, process maps, feature lists, mockups, visual specification, working prototypes and other artifacts to describe the intended user experience
  • Experience working with AJAX, XHTML, CSS, JQUERY, typography, various browser support
  • Experience with Drupal theme processing and customization
